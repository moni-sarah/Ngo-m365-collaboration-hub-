## Case Study: NGO M365 Collaboration Hub

> **5 SharePoint sites · 6 Teams workspaces · 0 CHF additional licensing · 5-week delivery**

**Client context:** Geneva-based international NGO, 40–80 staff, Microsoft 365 Nonprofit licence.

**Problem:** Fragmented communication across WhatsApp and personal email, 30+ ungoverned Teams workspaces, unstructured SharePoint with no findable content.

**Solution:** Hub-spoke SharePoint intranet + governed Teams architecture built entirely on existing M365 nonprofit licensing.

| Phase | Deliverable | Days |
|-------|-------------|------|
| 1 — Discovery | Governance register, content audit | 2d |
| 2 — IA + Teams | Site map, Teams policy, Azure AD groups | 3d |
| 3 — Build | 5 sites provisioned, permissions, metadata taxonomy | 4d |
| 4 — Content | Homepage, document migration, Viva Connections | 4d |
| 5 — Launch | Pilot, QA, all-staff launch | 2d |
